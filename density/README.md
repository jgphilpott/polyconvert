### [Density](https://gist.github.com/jgphilpott/4631fbcdbf42a612e46314b13d7bbc57)

The Density category provides 8 different units for conversion, see the object map below:

```js
convertDensity = {

    gramCentimeterCu: {...},
    gramMeterCu: {...},
    kilogramCentimeterCu: {...},
    kilogramMeterCu: {...},

    ounceInchCu: {...},
    ounceFootCu: {...},
    poundInchCu: {...},
    poundFootCu: {...}

}
```
