### [Frequency](https://gist.github.com/jgphilpott/cc4f9dac2a78b92928eadecc58394ab2)

The Frequency category provides 4 different units for conversion, see the object map below:

```js
polyconvert.frequency = {

    hertz: {...},
    kilohertz: {...},
    megahertz: {...},
    gigahertz: {...}

}
```
